## Cybersecurity Technical Skills Overview

After reading this article, you will be able to:

- Identify cybersecurity technical skills important for a cybersecurity professional.  
- Describe the concepts related to each skill.

| **Skill** | **Description** |
|----------|-----------------|
| **Information Security & Assurance** | • Security policies<br>• User access controls<br>• Conducting regular audits<br>• Ensuring compliance with policies |
| **Security Operations** | • Maintaining and monitoring security infrastructure<br>• Incident response activities<br>• Collaborating with IT teams to manage security incidents |
| **Cryptography** | • Cryptographic attacks<br>• Algorithms<br>• Public key infrastructure (PKI) |
| **Risk Management** | • Threat assessment<br>• Risk assessment<br>• Risk analysis |
| **Threat Management** | • Threat actors & attributes<br>• Threat investigation, detection & response<br>• Indicators of compromise (IOCs) |
| **Vulnerability Management** | • Penetration testing<br>• Vulnerability assessment<br>• Vulnerability scanning |
| **Authentication** | • Identity & access management controls<br>• Identity & access services<br>• Account management |
| **Digital Forensics** | • Chain of custody<br>• Data acquisition<br>• Data recovery & preservation |
